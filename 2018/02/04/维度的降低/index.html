维度的降低

1.PCA的理解

提起维度的降低，应用最多的便是PCA。对PCA的解释有两种：1种是最大化投影后数据的方差，因为方差最大的时候，数据的信息丢失的最少。第2种是最小化投影误差。2种方法最终都归结到求解到协方差矩阵的特征向量。

PCA的一般处理流程：

  • 1.将A的每一列按均值做归一化，使归一化后的A的每一列均值为0；
  • 2.求出A的协方差矩阵的特征值以及特征矩阵V。
  • 3.B = AV

PCA的一个优势是可以对变换后的样本数据进行重构只要保留特征矩阵就可以了。

2. K值的选择

一种确定K值的方法是K个特征值与原来D个特征值之和>=0.99或者0.95

3.对PCA的评价

  • 虽然能够降低维度，但是PCA不适合用来阻止过拟合，因为PCA没有考虑标签数据，只是考虑了样本数据的。
  • 一般求解方法使用SVD，速度更快。而且SVD没有计算协方差矩阵，对于某些矩阵，求协方差时很可能会丢失一些精度。
  • 一般情况下复杂度为O(D^3),但有一些学者的方法能达到O(MD^2)

    4.Kernel PCA

    传统的PCA只是利用线性的特征组合，如果想找到非线性的特征组合的话就要考虑Kenel PCA，它可以利用一些非线性核的性质。关于kernel PCA可参看kernel PCA。

    5.对kernel PCA的评价

  • Kernel PCA需要计算的矩阵的特征是NxN阶的，而线性PCA为DxD阶，对于高维数据，kernel trick依然可以使维度降至N
  • Kernel PCA能够提取出非线性的特征，具有更大的灵活性
