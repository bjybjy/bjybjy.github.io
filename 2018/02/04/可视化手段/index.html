可视化手段

python语言的可视化seaborn、pandas、matplotlib三大,相关的问题主要包括（在ipython notebook下）：

  • matplotlib.interactive(True)能够连续绘制弹出图片
  • 将图片嵌入ipython notebook:%pylab/matplotlib -inline
  • plt.rcParams[‘font.sans-serif’]=[‘simsun’] #将图片中所有字体改为宋体，用来正常显示中文标签simsun
  • plt.rcParams[‘axes.unicode_minus’]=False #用来正常显示负号
  • from matplotlib.font_manager import FontProperties
    font = FontProperties(fname=r”C:\WINDOWS\Fonts\simsun.ttc”, size=13)#C:\WINDOWS\Fonts
    plt.xlabel(x_lable, fontproperties=font)将x轴标签字体设为font
  • plt.legend(prop=font)将图例字体设置为font
  • ax=fig.add_subplot(111)
    ax.set_xlim(0,360)
    ax.set_xticks(ticks)设置刻度标签
  • plt.yticks(fontsize=13)设置刻度标签字体大小
  • matplotlib.style.use 选择配色模板（模板名称可以从plt.style.available）
